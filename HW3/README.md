## Usage

* Build with `Nova` Library.

```
./bin/HW3 -size 16 16
./bin/HW3 -size 32 32
./bin/HW3 -size 64 64
./bin/HW3 -size 128 128
./bin/HW3 -size 256 256
```
## Result

```

16	25		0.329824
32	51		0.312677
64	102		0.303788
128	207		0.299263
256	420		0.296981
```
